1.安装apache2
sudo apt-get apache2
重启Apache
sudo /etc/init.d/apache2 restart

2.安装mkdocs
sudo apt-get install python-pip (或在virtualenv中使用pip)
pip install mkdocs 
下面两个用于主题
pip install mkdocs-bootstrap
pip install mkdocs-bootswatch

3.制作文档工程框架
mkdocs new 工程名
查看服务
mkdocs serve
自动生成html文件
mkdocs build --clean
