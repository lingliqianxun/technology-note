安装
1.uname -a 查看内核版本和系统版本
2.sudo apt-get update
3.sudo apt-get install linux-image-generic-lts-trusty
4.sudo reboot
5.which wget (没有则安装sudo apt-get install wget)
6.wget -qO- https://get.docker.com/ | sh 安装或更新docker
7.sudo reboot 
8.sudo service docker status 查看是否有运行
9.sudo docker -d & 后台启动（阿里云sudo docker --bip=172.17.42.1/16 -d & ）
10.sudo docker run hello-world 测试（sudo usermod -aG docker user_name 可省略sudo)

问题一：出现FATA[0000] Error mounting devices cgroup: mountpoint for devices not found
重启sudo reboot
问题二：WARN[0000] Your kernel does not support swap memory limit. 
1.sudo vi /etc/default/grub
2.修改GRUB_CMDLINE_LINUX="cgroup_enable=memory swapaccount=1" (默认为"")
3.sudo update-grub
4.sudo reboot
问题三：如果UFW防火墙开启
1.sudo ufw status 如果安装且enabled
2.sudo vi /etc/default/ufw
3.修改DEFAULT_FORWARD_POLICY="ACCEPT" (默认为"DROP")
4.sudo ufw reload
5.sudo ufw allow 2375/tcp
问题四：WARNING: Local (127.0.0.1) DNS resolver found in resolv.conf and containers
can't use it. Using default external servers : [8.8.8.8 8.8.4.4]
1.sudo vi /etc/default/docker
2.添加DOCKER_OPTS="--dns 8.8.8.8 --dns 192.168.1.1"
3.sudo restart docker
或
1. sudo vi /etc/NetworkManager/NetworkManager.conf
2.修改dns=dnsmasq
3.sudo restart network-manager
4.sudo restart docker

问题五：FATA[0000] Error starting daemon: Error initializing network controller: Error creating default "bridge" network: can't find an address range for interface "docker0" 
因为阿里云默认把所有的私网地址段都加到导致docker找不到一个可用的IP段了。
1.ifconfig查看第一块网卡地址
2.sudo docker --bip=172.17.42.1/16 -d & 
或
1.vi /etc/default/docker
2.添加默认启动DOCKER_OPTS=\"-b=bridge0\或DOCKER_OPTS="--bip=192.168.17.1/24"

卸载
1.sudo apt-get purge lxc-docker
2.sudo apt-get autoremove --purge lxc-docker
3.rm -rf /var/lib/docker

命令
docker run [-d后台|-t伪终端|-i交互|-P随机端口映射|-p指定端口映射|--name容器名|-v数据卷|--volumes-from继承数据卷|--rm退出即删除|-y不提示|-net=host不使用bridge模式] 运行镜像
docker build [-t标签] 【Dockerfile_path】 创建镜像
docker commit [-m信息|-a用户] 【container_id】 【repository:tag】 提交镜像
docker tag 【image_id】 【repository:tag】 设置镜像标签
docker pull 【repository:tag】 下载镜像
docker push 【repository:tag】 向Docker Hub推送镜像
docker search 【repository:tag】 搜索可用镜像
docker images 查看本地镜像
docker rmi 【image_id】 删除本地镜像
docker save 【repository:tag】 > /home/xx.tar 保存镜像
docker load < /home/xx.tar 加载镜像

docker ps [-l状态] 列出容器
docker logs [-f前端|--since="2016-01-01T00:00:00"时间开始|-t显示时间|--tail=10末尾条数] 显示容器的标准输出
docker start 启动容器
docker stop 停止正在运行的容器
docker attach 连接容器
docker exec 容器执行命令
docker rm 删除容器
docker version 返回安装的docker客户端和进程信息
docker port 查询容器映射端口
docker top 查看容器进程
docker inspect 查看Docker底层信息
